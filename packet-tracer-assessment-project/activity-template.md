## Inside Packet Tracer

1. Open the source `.pkt`.
2. Save a working author copy.
3. Build the correct final configuration.
4. Open `Extension > Activity Wizard`.
5. Use the current workspace as the answer network.
6. Create or verify the initial network.
7. Add assessment items for:
   - device names
   - interfaces
   - IP addresses
   - VLANs
   - routing
   - ACLs
   - services
8. Add connectivity tests if needed.
9. Add instructions for the learner.
10. Save as `.pka`.

## Validation

- Opens correctly in Packet Tracer
- Instructions are visible
- Score changes when configuration is wrong
- Full score is reached when configuration is correct

## Notes

- Keep one clear grading objective per activity.
- Avoid mixing too many unrelated topics into one graded lab.
- Use separate `.pka` files instead of forcing everything into one master file.
